Enterprise-Grade Security

  • ✓E2EE Vault — Medical data encrypted with AES-256-GCM before it leaves the browser. Server never sees plaintext.
  • ✓Zero-Knowledge Proofs — Schnorr & snarkjs for privacy-preserving identity verification.
  • ✓JWT Rotation — 15-minute access tokens + 30-day sliding refresh via httpOnly, SameSite:Strict cookies.
  • ✓Rate Limiting — 500 req/min global cap + per-phone 60-second OTP throttle to prevent SMS wallet drain.
  • ✓DPDP Act Compliant — Patient consent flow, audit logs, and right-to-delete in place.
  • ✓Security Headers — CSP, HSTS, X-Frame-Options, Referrer-Policy applied globally.
  • ✓Banned User Guard — Every authenticated request checks ban status before any data access.
  • ✓Atomic DB Writes — ACID-compliant SQLite prevents data corruption under concurrent writes.
